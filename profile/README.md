## 🧩 What We Build

- **Identity & trust**：通行证、身份认证、KYC、权限与审核流。
- **Product tools**：抽奖、活动、统计、表单与后台管理系统。
- **Cloud services**：数据库、对象存储、人机验证与轻量级站点基础设施。
- **Open-source templates**：把可复用业务系统整理成小团队能读懂、能部署的项目。

## 🧭 Principles

- **Readable first** — 代码应当解释产品，而不是隐藏产品。
- **Secure by default** — 不提交密钥，使用环境变量配置，最小权限部署。
- **Small but complete** — 偏好可被小团队理解、fork 和部署的完整系统。
- **Docs before hype** — 每个公开项目都应包含安装、配置、部署与安全说明。
- **Open what is useful** — 开源不是堆仓库，而是沉淀真实可复用的工程经验。
